Locates a file adjacent to the current and initilizes the file pointer to reflect this accordingly.

Note
Uses the cdecl calling convention cdecl
Parameters
pRootFilePointer to the parent directory where its contained files will be enumerated
pFilePointer to the current file, or a root file descriptor to get the first file. This buffer is overwritten with the next file in the chain.
sFileNameAn optional pointer to a buffer where the name of the file is stored. Must be a minimum size of RSFS_NAME_SIZE. If this argument is zero, the name is not placed into the buffer pointed to by this address.
Returns
Returns a zero if the pFile is the last file in the list.
